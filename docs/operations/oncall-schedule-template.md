## 交接流程

### 每周一 00:00 交接时

**交接前 1 小时** (Primary 准备):
- [ ] 查看当前系统状态
- [ ] 确认无活跃事故
- [ ] 整理本周待办事项
- [ ] 准备交接信息

**交接时** (00:00):
1. **Primary 在 #on-call 群宣布交接**
2. **新 Primary 确认接班**
3. **新 Primary 查看监控 Dashboard**
4. **确认所有权限已配置**
5. **在 #on-call 群确认交接完成**

**交接后 15 分钟** (新 Primary 验证):
- [ ] 查看监控 Dashboard
- [ ] 确认告警通知正常
- [ ] 确认服务器访问正常
- [ ] 在 #on-call 群确认系统正常

### 交接模板

```
🔄 值班交接

时间: YYYY-MM-DD 00:00
交出: @ engineer-1
接手: @ engineer-2

系统状态: ✅ 正常 / ⚠️ 有问题
活跃事故: 无 / Issue #XXX
待办事项:
  - [ ] Issue #123 需跟进
  - [ ] 告警规则需优化

交接完成，@ engineer-2 开始值班

📊 快速检查:
- Dashboard: ✅ 正常
- 告警: ✅ 无活跃告警
- 服务器: ✅ 可访问
```
