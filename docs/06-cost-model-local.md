### 9.2 ROI计算

**保守估算**：
```
年度成本：¥148,600
年度收益：¥237,500
ROI = (237,500 - 148,600) / 148,600 = 59.8%
投资回收期：0.6年（7个月）
```

**合理估算**：
```
年度成本：¥200,000
年度收益：¥387,500
ROI = (387,500 - 200,000) / 200,000 = 93.8%
投资回收期：0.5年（6个月）
```

**乐观估算**：
```
年度成本：¥200,000
年度收益：¥787,500
ROI = (787,500 - 200,000) / 200,000 = 293.8%
投资回收期：0.3年（4个月）
```

**对比云方案ROI**：
| 方案 | 保守ROI | 合理ROI | 乐观ROI | 回收期 |
|------|---------|---------|---------|--------|
| 云方案 | -9.7% | 29% | 199% | 10个月 |
| **本地方案** | **59.8%** | **93.8%** | **293.8%** | **6个月** |

**提升**：ROI提升 70-200 个百分点，回收期缩短 40%
