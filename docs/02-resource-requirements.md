### 5.2 LLM API调用成本（DeepSeek）

**估算模型**：
- 50-200人规模，日均使用1000次
- 平均每次对话：1000 tokens输入 + 500 tokens输出

```
日均调用量：1000次
月度调用量：30,000次

DeepSeek API定价（估算）：
- 输入：¥0.001/1K tokens
- 输出：¥0.002/1K tokens

单次成本：
(1000 × 0.001 + 500 × 0.002) / 1000 = ¥0.002

月度成本：30,000 × ¥0.002 = ¥60
年度成本：¥60 × 12 = ¥720
```

**实际成本可能范围**：¥500-3,000/年
- 取决于使用频率和复杂度
- 高峰期可能达到5-10倍
