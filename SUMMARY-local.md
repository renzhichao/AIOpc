# AIOpc 项目总结（本地部署版）

## 🎉 项目概述

AIOpc是一个为电商公司设计的AI基础服务平台，**基于本地服务器部署OpenClaw开源框架**，为财务和电商部门提供智能化Agent服务能力。

## 💡 核心变更

### 从云部署到本地部署

| 方面 | 原方案（云为主） | **新方案（本地为主）** | 变化 |
|------|-----------------|---------------------|------|
| **部署位置** | 阿里云ECS（3台） | **本地服务器（4台）** | 降低81%基础设施成本 |
| **Agent运行** | 云端 | **本地内网** | 数据安全性更高 |
| **数据存储** | RDS云数据库 | **本地PostgreSQL** | 降低成本 |
| **访问方式** | 公网SLB | **内网/VPN** | 更安全的访问 |
| **年度成本** | ¥263,000-463,000 | **¥192,720-312,720** | **降低31-37%** |
| **ROI** | 29%（合理估算） | **93.8%** | **提升223%** |
| **回收期** | 10个月 | **6个月** | **缩短40%** |

## 🔧 关键配置

### 飞书集成（内网环境）

由于是内网环境，有以下方案：

**方案1：内网穿透**（测试用）
- 使用frp/ngrok
- 需要一台VPS（¥100-200/年）

**方案2：主动轮询**（推荐）
- Agent主动拉取消息
- 无需公网访问

**方案3：企业内部IM**（最佳）
- 企业微信/钉钉本地部署
- 完全内网化

### VPN配置

**WireGuard VPN**（推荐）：
```bash
# 安装
sudo apt-get install wireguard

# 配置
sudo wg genkey | sudo tee /etc/wireguard/privatekey
# 生成配置文件...

# 启动
sudo wg-quick up wg0
```

详细配置见：`docs/07-local-deployment-guide.md`
