| 功能 | 需求 | 当前实现 | GAP | 优先级 |

**GAP详情**:

```typescript
    model: 'deepseek-chat';

  skills: string[]; // ['general_chat', 'web_search', ...]
  tools: string[]; // ['read', 'write', 'web_search', ...]
  system_prompt: string; // 默认模板

// 新增端点
POST /api/instances
Response: {
  instance_id: string;
  preset_applied: true;
  ready_to_use: true
}
```

```
用户层 (飞书App, Web UI, API)
   ↓
网关层 (Nginx Ingress + SSL)
   ↓
应用层 (Web服务, OAuth服务, 实例管理)
   ↓
数据层 (PostgreSQL, Redis, OSS)
   ↓
实例层 (Docker容器)
```

```
✅ 前端: React + TypeScript
✅ API Gateway: Express + routing-controllers
✅ OAuth: Feishu OAuth 2.0

✅ 缓存: Redis
✅ 实例: Docker (本地) + 远程实例注册
```
