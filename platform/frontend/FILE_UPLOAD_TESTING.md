## 常见问题处理

### 问题1：上传后白屏
**处理步骤**：
1. 检查浏览器控制台（普通浏览器）
2. 启用调试面板（飞书WebView）
3. 查看window.__UPLOAD_DEBUG__
4. 复制日志分析问题

### 问题2：上传失败
**处理步骤**：
1. 查看错误提示
2. 检查文件大小和类型
3. 检查网络连接
4. 查看后端日志

### 问题3：进度不更新
**处理步骤**：
1. 检查是否真的在上传
2. 查看网络请求
3. 检查后端日志
4. 刷新页面重试
